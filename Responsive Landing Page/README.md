# Responsive Landing Page - Nexus

Welcome to a premium, responsive landing page demonstrating modern web design techniques. This project features an interactive navigation menu that adapts dynamically as you scroll or hover, utilizing glassmorphism, micro-animations, and a sleek dark mode aesthetic.

## Features

- **Interactive Navigation Menu**: The navbar has a fixed position and changes its background style to a glassmorphic look (blur + transparency) when scrolling down.
- **Hover Effects**: Menu items feature animated underlines and color transitions on hover.
- **Scroll Spy**: Navigation links automatically highlight based on the current section in view.
- **Premium Aesthetics**: Crafted with a carefully curated dark theme color palette, beautiful gradients, and modern typography (Google Fonts: Outfit & Inter).
- **Fully Responsive**: Adapts seamlessly from desktop to mobile screens, complete with a hamburger menu for smaller devices.
- **Micro-animations**: Elements like buttons, feature cards, and visuals include subtle hover and continuous animations (e.g., pulsing glow orb, morphing shapes) to make the page feel alive.

## Technology Stack

- **React**: Component-based UI architecture utilizing hooks (`useState`, `useEffect`) for scroll tracking and state management.
- **CSS3**: Advanced styling with CSS variables, Flexbox, Grid, animations, and backdrop-filter.
- **Vite**: Next-generation frontend tooling for an extremely fast development environment.

## Running Locally

To run this project on your local machine, follow these steps in your terminal:

1. **Install Dependencies**
   Ensure you have Node.js installed. Navigate to the project directory and run:
   ```bash
   npm install
   ```

2. **Start the Development Server**
   Run the following command to start Vite:
   ```bash
   npm run dev
   ```

3. **View in Browser**
   Open your browser and navigate to the URL provided in the terminal (usually `http://localhost:5173/`).

## Project Structure

- `index.html` - The main HTML file containing the structure of the landing page.
- `src/style.css` - The primary stylesheet containing all the design tokens, layouts, and animations.
- `src/main.js` - The JavaScript file handling the interactive behavior of the navigation menu.
