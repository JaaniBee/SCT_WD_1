# Pro Responsive Landing Page

A sleek, responsive, and fully functional landing page built with HTML, CSS, and JavaScript.

## Features

- **Interactive Navigation Menu**: The navbar has a fixed position and changes its background style to a glassmorphic look (blur + transparency) when scrolling down.
- **Hover Effects**: Menu items feature animated underlines and color transitions on hover.
- **Scroll Spy**: Navigation links automatically highlight based on the current section in view.
- **Interactive UI**: Smooth animations, intuitive controls, and a modern dark-mode aesthetic with glassmorphism touches.
- **Responsive Design**: Works flawlessly on desktop and mobile devices.

## Technologies Used

- **HTML5**: Semantic structure for accessibility and clarity.
- **CSS3**: Advanced styling with CSS variables, Flexbox, Grid, animations, and backdrop-filter.
- **JavaScript**: Interactive behavior and scroll tracking.
